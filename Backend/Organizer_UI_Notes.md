3.Events Management
    Show a table/list of events created by the organizer.
        Columns example:
            Event Name
            Date
            Location
            Registrations
            Status
            Actions

        Actions:
            Create Event
            ✏ Edit Event
            🗑 Delete Event
            👁 View Details
            📊 View Participants


4. Participants Management

    Organizer should be able to:
        See who registered
        Download participant list
        Approve / reject participants
        Send event updates

    Columns example:
        Name
        Email
        Phone
        College
        Status (Approved / Pending)
    Extra feature:
        Export to CSV or Excel.

6. Notifications / Announcements
    Organizer can send updates to participants.
        Examples:
            Event time changed
            Venue updated
            Reminder notifications
